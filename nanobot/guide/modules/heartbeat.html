心跳服务 (Heartbeat) ​

定期触发的任务机制

版本: v1.0.0 | 代码版本: nanobot v0.1.4.post4 | 更新日期: 2026-03-12

模块概述 ​

职责 ​

Heartbeat 服务负责：

  1. 定期触发：按固定间隔执行任务
  2. 任务加载：从 HEARTBEAT.md 加载任务
  3. 时间计算：计算下次触发时间
  4. 状态管理：跟踪运行状态

与 Cron 的区别 ​

特性HeartbeatCron
调度方式固定间隔灵活表达式
配置文件HEARTBEAT.mdjobs.json
任务类型周期性任务一次性 + 周期性
适用场景简单定期任务复杂调度需求

HeartbeatService 类 ​

python
class HeartbeatService:
    """心跳服务"""

    def __init__(self):
        self._running = False
        self._task: asyncio.Task | None = None

    async def start(
        self,
        callback: Callable,
        agent: AgentLoop,
    ):
        """
        启动心跳

        Args:
            callback: 触发时的回调函数
            agent: Agent 实例
        """
        self._running = True

        # 加载 HEARTBEAT.md
        tasks = self._load_heartbeat_tasks()

        # 计算下次触发时间
        next_time = self._calculate_next_time(tasks)

        # 启动定时器
        self._task = asyncio.create_task(
            self._heartbeat_loop(callback, agent, tasks, next_time)
        )

    async def stop(self):
        """停止心跳"""
        self._running = False
        if self._task:
            self._task.cancel()

时间计算 ​

python
def _calculate_next_time(self, tasks: dict) -> datetime:
    """
    计算下次触发时间

    every: 1h -> 每小时整点
    every: 30m -> 每半小时
    every: 1d -> 每天 0 点（或指定 at 时间）
    """
    if "at" in tasks:
        # 使用指定时间
        hour, minute = parse_time(tasks["at"])
        next_run = datetime.now().replace(
            hour=hour,
            minute=minute,
            second=0,
            microsecond=0
        )
        if next_run < datetime.now():
            next_run += timedelta(days=1)
    else:
        # 使用间隔
        interval = parse_interval(tasks.get("every", "1h"))
        next_run = datetime.now() + interval

    return next_run

async def _heartbeat_loop(self, callback, agent, tasks, next_time):
    """心跳循环"""
    while self._running:
        now = datetime.now()

        if now >= next_time:
            # 触发任务
            await callback(agent, tasks)

            # 计算下次时间
            next_time = self._calculate_next_time(tasks)

        # 等待一段时间再检查
        await asyncio.sleep(60)

3. 执行任务 ​

async def _heartbeat_callback(agent: AgentLoop, tasks: dict):
    """心跳回调"""
    task_description = tasks.get("content", "")

    # 创建系统消息
    msg = InboundMessage(
        channel="system",
        sender_id="heartbeat",
        chat_id="cli",
        content=task_description,
        metadata={"heartbeat": True}
    )

    # 处理消息
    await agent._process_message(msg)

常见问题 ​

Q1: Heartbeat 和 Cron 如何选择？ ​

A:

  • Heartbeat: 简单的固定间隔任务
  • Cron: 复杂的调度需求

Q2: 如何禁用 Heartbeat？ ​

A: 删除或重命名 HEARTBEAT.md

Q3: 多个 HEARTBEAT 任务？ ​

A: 当前只支持一个，如需多个使用 Cron
