聊天通道系统 (Channels) ​

连接不同聊天平台的适配层

版本: v1.0.0 | 代码版本: nanobot v0.1.4.post4 | 更新日期: 2026-03-12

模块概述 ​

职责 ​

Channels 负责连接不同的聊天平台：

  1. 接收消息：从平台接收用户消息
  2. 发送响应：向平台发送 AI 响应
  3. 权限控制：检查用户访问权限
  4. 格式转换：转换平台特定格式

BaseChannel 抽象类 ​

python
class BaseChannel(ABC):
    """聊天通道抽象基类"""

    @property
    @abstractmethod
    def name(self) -> str:
        """通道名称"""
        pass

    @abstractmethod
    async def start(self):
        """启动通道"""
        pass

    @abstractmethod
    async def stop(self):
        """停止通道"""
        pass

    @abstractmethod
    async def send(self, msg: OutboundMessage):
        """发送消息"""
        pass

    def is_allowed(self, sender_id: str) -> bool:
        """检查权限"""
        pass

class MyCustomChannel(BaseChannel):
    def __init__(self, config: dict, bus: MessageBus):
        self.config = config
        self.bus = bus
        self._running = False

    @property
    def name(self) -> str:
        return "myplatform"

    async def start(self):
        """启动通道"""
        self._running = True
        # 连接平台，监听消息

    async def stop(self):
        """停止通道"""
        self._running = False
        # 断开连接

    async def send(self, msg: OutboundMessage):
        """发送消息到平台"""
        # 发送 msg.content 到 msg.chat_id
        pass

    def is_allowed(self, sender_id: str) -> bool:
        """检查权限"""
        allow_from = self.config.get("allowFrom", ["*"])
        return "*" in allow_from or sender_id in allow_from

步骤 3: 处理入站消息 ​

python
async def _handle_message(self, platform_msg):
    """处理平台消息"""

    # 1. 检查权限
    if not self.is_allowed(platform_msg.sender_id):
        return

    # 2. 转换为 InboundMessage
    inbound = InboundMessage(
        channel=self.name,
        sender_id=platform_msg.sender_id,
        chat_id=platform_msg.chat_id,
        content=platform_msg.text,
        media=platform_msg.attachments,
        metadata={"message_id": platform_msg.id}
    )

    # 3. 发布到消息总线
    await self.bus.publish_inbound(inbound)

ChannelManager ​

python
class ChannelManager:
    """通道管理器"""

    def __init__(self, config: Config, bus: MessageBus):
        self.config = config
        self.bus = bus
        self.channels: list[BaseChannel] = []

    async def start(self):
        """启动所有启用的通道"""
        for name in self.config.enabled:
            channel = self._create_channel(name)
            if channel:
                await channel.start()
                self.channels.append(channel)

    def _create_channel(self, name: str) -> BaseChannel:
        """创建通道实例"""
        if name == "telegram":
            return TelegramChannel(
                self.config.telegram,
                self.bus
            )
        elif name == "discord":
            return DiscordChannel(
                self.config.discord,
                self.bus
            )

权限配置 ​

配置格式 ​

json
{
  "telegram": {
    "token": "your-bot-token",
    "allowFrom": ["123456789", "987654321"]
  },
  "discord": {
    "token": "your-bot-token",
    "allowFrom": ["*"]
  }
}

权限规则 ​

配置效果
"allowFrom": ["*"]允许所有用户
"allowFrom": ["123"]只允许用户 123
"allowFrom": []拒绝所有用户
