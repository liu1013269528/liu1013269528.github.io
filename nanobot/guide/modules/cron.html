定时任务服务 (Cron) ​

基于时间的任务调度系统

版本: v1.0.0 | 代码版本: nanobot v0.1.4.post4 | 更新日期: 2026-03-12

模块概述 ​

职责 ​

Cron 服务负责：

  1. 任务调度：按时间规则执行任务
  2. 任务持久化：保存任务配置
  3. 任务执行：触发时调用回调
  4. 任务管理：创建、删除、列出任务

支持的调度类型 ​

类型说明示例
at一次性任务在指定时间执行
every间隔重复每 5 分钟执行一次
cronCron 表达式每天 9:00 执行

CronService 类 ​

python
class CronService:
    """定时任务服务"""

    def __init__(self, store_path: Path):
        self.store_path = store_path  # jobs.json
        self.jobs: dict[str, CronJob] = {}
        self._running = False

    async def start(self):
        """启动调度器"""

    async def create(
        self,
        task: str,
        schedule: str,
        schedule_type: str = "every",
    ) -> str:
        """
        创建定时任务

        Returns:
            job_id: 任务 ID
        """

    async def delete(self, job_id: str) -> bool:
        """删除任务"""

    async def list_jobs(self) -> list[CronJob]:
        """列出所有任务"""

    async def _scheduler_loop(self):
        """调度循环（每秒检查）"""

CronJob ​

python
@dataclass
class CronJob:
    """定时任务"""
    id: str                    # 任务 ID
    task: str                  # 任务描述
    schedule: str              # 调度规则
    schedule_type: str         # at/every/cron
    created_at: datetime       # 创建时间
    next_run: datetime | None  # 下次运行时间
    last_run: datetime | None  # 上次运行时间

调度类型 ​

1. At - 一次性任务 ​

python
# 在 2026-03-12 15:30 执行
await cron.create(
    task="发送下午报告",
    schedule="2026-03-12 15:30",
    schedule_type="at"
)

2. Every - 间隔重复 ​

python
# 每 5 分钟执行一次
await cron.create(
    task="检查邮件",
    schedule="5m",
    schedule_type="every"
)

# 每 1 小时执行一次
await cron.create(
    task="备份数据",
    schedule="1h",
    schedule_type="every"
)

3. Cron - Cron 表达式 ​

python
# 每天上午 9 点执行
await cron.create(
    task="每日汇总",
    schedule="0 9 * * *",
    schedule_type="cron"
)

# Cron 格式：分 时 日 月 周
# 0 9 * * * = 每天 9:00
# */5 * * * * = 每 5 分钟
# 0 */2 * * * = 每 2 小时

Cron 工具 ​

使用方式 ​

用户：创建一个定时任务，每天早上 9 点发送天气报告

LLM：我会创建一个定时任务。

[工具调用] cron(
    task="每天早上 9 点发送天气报告",
    schedule="0 9 * * *",
    schedule_type="cron"
)

系统：✓ 已创建定时任务，ID：job_abc123

LLM：定时任务已创建。每天早上 9 点会自动发送天气报告。

常见问题 ​

Q1: 如何列出所有任务？ ​

A: 使用 cron 工具不带参数，或调用 list_jobs()

Q2: 任务失败会重试吗？ ​

A: 不会，但会记录错误

Q3: 如何删除任务？ ​

A: 使用 cron 工具的 delete 操作
