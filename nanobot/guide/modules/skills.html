技能系统 (Skills) ​

用 Markdown 教 AI 如何完成任务

版本: v1.0.0 | 代码版本: nanobot v0.1.4.post4 | 更新日期: 2026-03-12

模块概述 ​

职责 ​

Skills 系统负责：

  1. 加载技能文件：从 workspace/skills/ 读取
  2. 管理内置技能：nanobot/skills/ 中的默认技能
  3. 技能激活：Always 技能自动激活
  4. 技能摘要：为 LLM 提供技能列表

技能文件格式 ​

markdown
---
name: commit
description: Git commit 助手
always: false
dependencies: []
---

# Git Commit 助手

这个技能帮助创建规范的 Git commit 消息。

## 使用方法

1. 说明修改的内容
2. AI 会生成 commit 消息
3. 确认后执行 `git commit`

python
class SkillsLoader:
    """技能加载器"""

    def __init__(self, workspace: Path):
        self.workspace = workspace
        self.user_skills_dir = workspace / "skills"
        self.builtin_skills_dir = Path(__file__).parent.parent / "skills"

    def get_always_skills(self) -> list[str]:
        """获取 always 技能列表"""

    def load_skills_for_context(self, skill_names: list[str]) -> str:
        """加载技能内容到上下文"""

    def build_skills_summary(self) -> str:
        """构建技能摘要表格"""

技能类型 ​

1. Always 技能 ​

markdown
---
always: true
---

# 始终激活

这个技能会在每次对话时自动加载

2. 按需技能 ​

markdown
---
always: false
---

# 按需激活

LLM 需要时读取 SKILL.md

技能位置优先级 ​

1. Workspace 技能（用户自定义）
   ~/.nanobot/workspace/skills/

2. 内置技能（默认）
   nanobot/skills/

如果有重名，workspace 技能优先

内置技能 ​

技能描述位置
commitGit commit 助手nanobot/skills/commit/
prPR 创建助手nanobot/skills/pr/
review代码审查nanobot/skills/review/

创建自定义技能 ​

步骤 1: 创建技能目录 ​

bash
mkdir -p ~/.nanobot/workspace/skills/my-skill

步骤 2: 创建 SKILL.md ​

bash
vim ~/.nanobot/workspace/skills/my-skill/SKILL.md
markdown
---
name: my-skill
description: 我的自定义技能
always: false
dependencies: []
---

# 我的自定义技能

这个技能做什么...

常见问题 ​

Q1: 技能如何激活？ ​

A: Always 技能自动激活，其他技能 LLM 按需读取

Q2: 如何列出所有技能？ ​

A: 技能摘要会显示在系统提示词中

Q3: 技能可以调用工具吗？ ​

A: 可以，技能是教 AI 如何使用工具的指南
