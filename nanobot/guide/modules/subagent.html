子代理管理 (Subagent) ​

后台独立任务管理

版本: v1.0.0 | 代码版本: nanobot v0.1.4.post4 | 更新日期: 2026-03-12

模块概述 ​

职责 ​

Subagent 系统负责：

  1. 创建子代理：在后台运行独立任务
  2. 任务追踪：管理和监控子代理
  3. 结果收集：获取子代理的执行结果
  4. 资源清理：完成后清理资源

使用场景 ​

  • 长时间运行的任务
  • 独立的研究任务
  • 并行处理多个请求
  • 不阻塞主对话

SubagentManager 类 ​

python
class SubagentManager:
    """子代理管理器"""

    def __init__(
        self,
        provider: LLMProvider,
        workspace: Path,
        bus: MessageBus,
        model: str,
        temperature: float,
        ...
    ):
        self.provider = provider
        self.workspace = workspace
        self.bus = bus
        self.subagents: dict[str, AgentLoop] = {}

    async def spawn(
        self,
        task: str,
        session_key: str,
        on_progress: Callable | None = None,
    ) -> str:
        """
        创建子代理

        Returns:
            subagent_id: 子代理 ID
        """

    async def get_result(self, subagent_id: str) -> str | None:
        """获取子代理结果"""

    async def cancel_by_session(self, session_key: str) -> int:
        """取消会话的所有子代理"""

Spawn 工具 ​

使用方式 ​

用户：帮我研究一下 Python 3.12 的新特性

LLM：我会创建一个后台任务来研究这个。

[工具调用] spawn(task="研究 Python 3.12 新特性")

系统：已创建后台任务，ID 是 task_abc123

LLM：我正在后台研究 Python 3.12 的新特性。
完成后会通知你。

[一段时间后]

系统：任务 task_abc123 完成
结果：Python 3.12 的主要新特性包括...

工作流程 ​

1. 创建子代理 ​

python
# 用户触发 spawn 工具
result = await registry.execute("spawn", {
    "task": "研究某个主题"
})

# SubagentManager.spawn()
# 1. 创建新的 AgentLoop
# 2. 创建独立的会话
# 3. 在后台运行任务
# 4. 返回 subagent_id

2. 监控进度 ​

python
# 子代理运行时可以发送进度更新
await on_progress("正在收集信息...")
await on_progress("正在分析数据...")

3. 获取结果 ​

python
# 任务完成后获取结果
result = await subagent_manager.get_result(subagent_id)

4. 清理资源 ​

python
# 自动清理完成的子代理
# 或手动取消
await subagent_manager.cancel(subagent_id)

配置 ​

工具配置 ​

python
class SpawnTool(Tool):
    def __init__(self, manager: SubagentManager):
        self.manager = manager

    async def execute(self, task: str) -> str:
        subagent_id = await self.manager.spawn(
            task=task,
            session_key=self.session_key,
            on_progress=self.progress_callback
        )
        return f"已创建后台任务：{subagent_id}"

常见问题 ​

Q1: 子代理和主代理有什么区别？ ​

A:

  • 子代理：独立会话，后台运行，不阻塞主对话
  • 主代理：用户会话，交互式，响应用户

Q2: 如何查看子代理状态？ ​

A: 子代理会发送进度更新

Q3: 子代理会占用资源吗？ ​

A: 会，完成后自动清理
