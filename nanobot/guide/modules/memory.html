记忆系统 (Memory) ​

nanobot 的长期记忆管理

版本: v1.0.0 | 代码版本: nanobot v0.1.4.post4 | 更新日期: 2026-03-12

模块概述 ​

职责 ​

Memory 系统负责：

  1. 存储重要信息：用户偏好、重要事实
  2. 历史日志：按时间记录的对话摘要
  3. 记忆合并：将长对话压缩为记忆
  4. 记忆检索：为 LLM 提供相关记忆

文件结构 ​

~/.nanobot/workspace/memory/
├── MEMORY.md      # 长期记忆（重要事实）
└── HISTORY.md     # 历史日志（时间顺序）

markdown

合并步骤 ​

python
async def _consolidate_memory(self, session: Session) -> bool:
    """
    记忆合并流程

    1. 获取未合并的消息
    2. 调用 LLM 生成摘要
    3. 更新 MEMORY.md
    4. 追加到 HISTORY.md
    5. 更新 last_consolidated
    """
    # 1. 获取消息
    messages = session.messages[session.last_consolidated:]

    # 2. 生成摘要
    summary = await self._generate_summary(messages)

    # 3. 更新 MEMORY.md
    self.memory.append_to_memory(summary)

    # 4. 追加到 HISTORY.md
    timestamp = datetime.now().strftime("%Y-%m-%d %H:%M")
    history_entry = f"## [{timestamp}] {summary['title']}\n\n{summary['content']}"
    self.memory.append_to_history(history_entry)

    # 5. 更新标记
    session.last_consolidated = len(session.messages)

    return True

MemoryStore 类 ​

python
class MemoryStore:
    """记忆存储"""

    def __init__(self, workspace: Path):
        self.workspace = workspace
        self.memory_file = workspace / "memory" / "MEMORY.md"
        self.history_file = workspace / "memory" / "HISTORY.md"

    def get_memory_context(self) -> str:
        """获取记忆内容（用于 LLM）"""
        if self.memory_file.exists():
            return self.memory_file.read_text(encoding="utf-8")
        return ""

    def append_to_memory(self, content: str):
        """追加到 MEMORY.md"""
        self.memory_file.parent.mkdir(parents=True, exist_ok=True)
        with open(self.memory_file, "a", encoding="utf-8") as f:
            f.write(f"\n{content}\n")

    def append_to_history(self, content: str):
        """追加到 HISTORY.md"""
        self.history_file.parent.mkdir(parents=True, exist_ok=True)
        with open(self.history_file, "a", encoding="utf-8") as f:
            f.write(f"\n{content}\n")

常见问题 ​

Q1: 记忆会自动清理吗？ ​

A: 不会，记忆是持久化的，除非手动删除

Q2: 如何编辑记忆？ ​

A: 直接编辑 MEMORY.md 文件

Q3: 记忆合并会消耗 token 吗？ ​

A: 会，但合并后历史消息不会重复发送，节省 token

Q4: 如何禁用记忆合并？ ​

A: 设置很大的 memory_window
