模块概述 ​

职责 ​

工具系统负责：

  1. 工具注册：管理所有可用工具
  2. 参数验证：验证工具调用参数
  3. 工具执行：安全地执行工具
  4. 结果返回：将结果格式化后返回给 LLM

设计模式 ​

1. 命令模式 (Command Pattern) ​

定义：将请求封装为对象

在工具系统中的应用：

python
class Tool(ABC):
    """工具基类 - 封装可执行的操作"""

    @abstractmethod
    async def execute(self, **kwargs) -> str:
        """执行工具命令"""
        pass

# 每个工具都是一个命令
class ReadFileTool(Tool):
    async def execute(self, path: str) -> str:
        """执行读取文件命令"""
        return file_content

好处：

  • ✅ 统一的接口
  • ✅ 可以延迟执行
  • ✅ 易于扩展

2. 注册表模式 (Registry Pattern) ​

定义：集中管理和查找可扩展组件

在工具系统中的应用：

python
class ToolRegistry:
    """工具注册表"""

    def __init__(self):
        self._tools: dict[str, Tool] = {}

    def register(self, tool: Tool):
        """注册工具"""
        self._tools[tool.name] = tool

    def get(self, name: str) -> Tool:
        """获取工具"""
        return self._tools.get(name)

好处：

  • ✅ 动态注册和发现
  • ✅ 解耦工具定义和使用
  • ✅ 易于管理

核心组件 ​

Tool 基类 ​

文件: nanobot/agent/tools/base.py

python
class Tool(ABC):
    """
    工具抽象基类

    所有工具必须继承此类并实现抽象方法
    """

    # 类型映射（用于参数转换）
    _TYPE_MAP = {
        "string": str,
        "integer": int,
        "number": (int, float),
        "boolean": bool,
        "array": list,
        "object": dict,
    }

    @property
    @abstractmethod
    def name(self) -> str:
        """工具名称 - 用于函数调用"""
        pass

    @property
    @abstractmethod
    def description(self) -> str:
        """工具描述 - 告诉 LLM 这个工具做什么"""
        pass

    @property
    @abstractmethod
    def parameters(self) -> dict[str, Any]:
        """参数 JSON Schema - 定义工具的参数"""
        pass

    @abstractmethod
    async def execute(self, **kwargs: Any) -> str:
        """
        执行工具

        Returns:
            字符串格式的结果（发送给 LLM）
        """
        pass

ToolRegistry 注册表 ​

文件: nanobot/agent/tools/registry.py

python
class ToolRegistry:
    """工具注册表"""

    def __init__(self):
        self._tools: dict[str, Tool] = {}

    # 注册和注销
    def register(self, tool: Tool) -> None:
        """注册工具"""
        self._tools[tool.name] = tool

    def unregister(self, name: str) -> None:
        """注销工具"""
        self._tools.pop(name, None)

    # 查询
    def get(self, name: str) -> Tool | None:
        """获取工具"""
        return self._tools.get(name)

    def has(self, name: str) -> bool:
        """检查工具是否存在"""
        return name in self._tools

    @property
    def tool_names(self) -> list[str]:
        """所有工具名称"""
        return list(self._tools.keys())

    # 获取定义（用于 LLM）
    def get_definitions(self) -> list[dict]:
        """获取所有工具的 OpenAI 格式定义"""
        return [tool.to_schema() for tool in self._tools.values()]

    # 执行工具
    async def execute(self, name: str, params: dict) -> str:
        """
        执行工具

        1. 查找工具
        2. 类型转换
        3. 参数验证
        4. 执行工具
        5. 返回结果
        """
        tool = self._tools.get(name)
        if not tool:
            return f"Error: Tool '{name}' not found"

        # 参数类型转换
        params = tool.cast_params(params)

        # 参数验证
        errors = tool.validate_params(params)
        if errors:
            return f"Error: Invalid parameters: {'; '.join(errors)}"

        # 执行工具
        try:
            result = await tool.execute(**params)
            return result
        except Exception as e:
            return f"Error executing {name}: {str(e)}"

创建复杂工具 ​

python
from pathlib import Path

class ReadFileTool(Tool):
    """读取文件工具"""

    def __init__(self, workspace: Path, allowed_dir: Path | None = None):
        self.workspace = workspace
        self.allowed_dir = allowed_dir

    @property
    def name(self) -> str:
        return "read_file"

    @property
    def description(self) -> str:
        return "读取指定路径的文件内容"

    @property
    def parameters(self) -> dict:
        return {
            "type": "object",
            "properties": {
                "path": {
                    "type": "string",
                    "description": "要读取的文件路径"
                }
            },
            "required": ["path"]
        }

    async def execute(self, path: str) -> str:
        # 1. 解析路径
        full_path = self._resolve_path(path)

        # 2. 检查权限
        if not self._is_allowed(full_path):
            return f"Error: 拒绝访问 {path}"

        # 3. 读取文件
        try:
            with open(full_path, 'r', encoding='utf-8') as f:
                content = f.read()
            return content
        except FileNotFoundError:
            return f"Error: 文件不存在: {path}"
        except Exception as e:
            return f"Error: 读取文件失败: {str(e)}"

    def _resolve_path(self, path: str) -> Path:
        """解析路径"""
        p = Path(path)
        if not p.is_absolute():
            p = self.workspace / p
        return p.resolve()

    def _is_allowed(self, path: Path) -> bool:
        """检查路径是否允许访问"""
        if self.allowed_dir is None:
            return True
        try:
            path.resolve().relative_to(self.allowed_dir.resolve())
            return True
        except ValueError:
            return False

参数验证 ​

python
class CalculatorTool(Tool):
    """计算器工具"""

    @property
    def name(self) -> str:
        return "calculate"

    @property
    def description(self) -> str:
        return "执行数学计算"

    @property
    def parameters(self) -> dict:
        return {
            "type": "object",
            "properties": {
                "expression": {
                    "type": "string",
                    "description": "数学表达式，如 '2 + 2'",
                    "minLength": 1,
                    "maxLength": 100
                },
                "precision": {
                    "type": "integer",
                    "description": "小数位数",
                    "minimum": 0,
                    "maximum": 10,
                    "default": 2
                }
            },
            "required": ["expression"]
        }

    async def execute(self, expression: str, precision: int = 2) -> str:
        try:
            # 安全地评估表达式
            result = eval(expression, {"__builtins__": {}}, {})
            return f"{result:.{precision}f}"
        except Exception as e:
            return f"Error: 无效的表达式: {str(e)}"

验证示例：

python
# 有效参数
tool.validate_params({"expression": "2 + 2"})  # []

# 缺少必需参数
tool.validate_params({})  # ["missing required expression"]

# 类型错误
tool.validate_params({"expression": 123})  # ["expression should be string"]

# 范围错误
tool.validate_params({"expression": "2+2", "precision": 15})  # ["precision must be <= 10"]

常见问题 ​

Q1: 工具参数如何传递？ ​

A: 通过 JSON 格式的字典：

python
# LLM 调用工具
params = {
    "path": "/path/to/file.txt",
    "encoding": "utf-8"
}

# 注册表执行
result = await registry.execute("read_file", params)

# Tool.execute() 被调用
# await execute(path="/path/to/file.txt", encoding="utf-8")

Q2: 如何处理工具错误？ ​

A: 返回错误消息：

python
async def execute(self, ...) -> str:
    try:
        # 工具逻辑
        result = do_something()
        return result
    except ValueError as e:
        return f"Error: 参数错误: {str(e)}"
    except Exception as e:
        return f"Error: 执行失败: {str(e)}"

注意：错误消息会返回给 LLM，LLM 可能会尝试其他方法。

Q3: 工具可以调用其他工具吗？ ​

A: 可以，但需要注入注册表：

python
class CompositeTool(Tool):
    def __init__(self, registry: ToolRegistry):
        self.registry = registry

    async def execute(self, ...) -> str:
        # 调用其他工具
        result1 = await self.registry.execute("tool1", {...})
        result2 = await self.registry.execute("tool2", {...})
        return f"{result1} + {result2}"

Q4: 如何限制工具的访问权限？ ​

A: 在工具内部检查：

python
class SecureTool(Tool):
    def __init__(self, allowed_paths: list[Path]):
        self.allowed_paths = allowed_paths

    async def execute(self, path: str) -> str:
        # 检查路径是否允许
        if not self._is_allowed(path):
            return "Error: 权限拒绝"
        # 执行操作
        return do_operation(path)

    def _is_allowed(self, path: str) -> bool:
        p = Path(path)
        return any(
            p.resolve().is_relative_to(allowed.resolve())
            for allowed in self.allowed_paths
        )

Q5: 工具支持异步吗？ ​

A: 是的，所有工具都是异步的：

python
class AsyncTool(Tool):
    async def execute(self, url: str) -> str:
        # 异步 HTTP 请求
        async with aiohttp.ClientSession() as session:
            async with session.get(url) as resp:
                return await resp.text()

# 调用
result = await registry.execute("fetch", {"url": "https://example.com"})
