快速开始 ​

安装 ​

bash
pip install -e .

配置 ​

bash
nanobot onboard

这将创建配置文件和工作空间。

配置 API Key ​

编辑 ~/.nanobot/config.json，添加你的 API Key：

json
{
  "providers": {
    "anthropic": {
      "apiKey": "sk-ant-..."
    }
  }
}

运行 ​

bash
# 直接对话
nanobot agent -m "你好！"

# 启动网关
nanobot gateway
