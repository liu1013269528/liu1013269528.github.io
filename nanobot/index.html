核心架构 ​

聊天平台 → Channel → MessageBus → AgentLoop → LLM + Tools
   ↑                                              ↓
   └────────────────响应──────────────────────────┘

核心组件 ​

  • MessageBus: 消息队列，解耦通道和代理
  • AgentLoop: 核心处理引擎，处理消息和工具调用
  • ChannelManager: 管理所有聊天通道
  • ToolRegistry: 管理所有可用工具
  • SessionManager: 管理对话会话和历史

技术栈 ​

  • Python 3.11+
  • asyncio: 异步编程
  • Pydantic: 数据验证
  • Typer: CLI 框架
  • VitePress: 文档生成
